Direct Plagiarism
Definition
Direct Plagiarism (also called verbatim plagiarism) is when an entire line or sentence from one document appears exactly as-is in another document — word for word, with no changes.
The algorithm reads both files line by line, converts everything to lowercase, then searches for lines in Document A that appear identically in Document B. The percentage is calculated as:

Score = (Matched Lines ÷ Total Lines in A) × 100
Example
Document A
the sky is blue today
birds fly very high
water is essential for life
Document B
the sky is blue today
clouds form in the air
water is essential for life
2 out of 3 lines match → Direct Plagiarism = 66.67%
When is it flagged?
Even a single word difference (punctuation, spelling, extra space) will cause a line to not match. Direct plagiarism only catches copy-paste with zero modification.
Partial Plagiarism
Definition
Partial Plagiarism detects shared vocabulary — it checks how many individual words from Document A can be found anywhere in Document B. This catches paraphrased or reworded content.
Every word in Document A is checked against the full word list of Document B. If a word exists anywhere in B, it's counted as a match.

Score = (Matched Words ÷ Total Words in A) × 100
Example
Document A words
the sky is cloudy today
Document B words
the sky is blue today above us
4 out of 5 words match → Partial Plagiarism = 80.00%
(even though the lines are different!)
Why does this matter?
Writers often paraphrase — changing sentence order or replacing a few words — to disguise copied content. Partial plagiarism catches this by focusing on what words are used, not whether lines are identical.
Logic Source Code — C++
The C++ class below is the original logic this web tool is based on. The JavaScript implementation mirrors these exact algorithms.
// Plagiarism Checker — Core Logic (C++) #include <iostream> #include <fstream> #include <vector> #include <string> #include <sstream> class PlagiarismChecker { private: std::vector<std::string> lines1, lines2; std::vector<std::string> words1, words2; // Convert string to lowercase void toLower(std::string &s) { for (char &c : s) if (c >= 'A' && c <= 'Z') c = c + 32; } // Read file lines into vector void readLines(const std::string &file, std::vector<std::string> &lines) { std::ifstream fin(file); std::string line; if (!fin.is_open()) { std::cout << "Error opening file: " << file << std::endl; return; } while (std::getline(fin, line)) { toLower(line); if (!line.empty()) lines.push_back(line); } fin.close(); } // Extract words from lines void extractWords(const std::vector<std::string> &lines, std::vector<std::string> &words) { for (const std::string &line : lines) { std::stringstream ss(line); std::string word; while (ss >> word) words.push_back(word); } } public: void loadFiles(const std::string &f1, const std::string &f2) { readLines(f1, lines1); readLines(f2, lines2); extractWords(lines1, words1); extractWords(lines2, words2); } // Line-by-line exact matching double detectDirectPlagiarism() { int matchCount = 0, j = 0; for (int i = 0; i < lines1.size(); i++) { while (j < lines2.size()) { if (lines1[i] == lines2[j]) { matchCount++; j++; break; } j++; } } if (lines1.size() == 0) return 0.0; return (double)matchCount / lines1.size() * 100; } // Word-level matching (shared vocabulary) double detectPartialPlagiarism() { int matchCount = 0; for (int i = 0; i < words1.size(); i++) { for (int j = 0; j < words2.size(); j++) { if (words1[i] == words2[j]) { matchCount++; break; } } } if (words1.size() == 0) return 0.0; return (double)matchCount / words1.size() * 100; } }; int main() { std::string file1, file2; std::cout << "Enter first file name: "; std::cin >> file1; std::cout << "Enter second file name: "; std::cin >> file2; PlagiarismChecker checker; checker.loadFiles(file1, file2); double direct = checker.detectDirectPlagiarism(); double partial = checker.detectPartialPlagiarism(); double similarity = (direct > partial) ? direct : partial; std::cout << "\nDirect Plagiarism: " << direct << "%\n"; std::cout << "Partial Plagiarism: " << partial << "%\n"; std::cout << "Similarity: " << similarity << "%\n"; if (similarity < 10) std::cout << "Very Low Plagiarism\n"; else if (similarity < 20) std::cout << "Low Plagiarism\n"; else if (similarity < 40) std::cout << "High Plagiarism\n"; else if (similarity < 70) std::cout << "Very High Plagiarism\n"; else std::cout << "Extreme Plagiarism\n"; return 0; }
